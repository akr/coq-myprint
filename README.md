# myprint plugin for Coq

This software provides Coq commands to print Gallina terms.

## Requiements

- Coq 8.10+beta3 (Coq 8.9 doesn't work)
- OCaml 4.09.0

## How to build

    make

## How to run

    coqide sample/print.v

## How to use

See sample/ directory.
